Detecting if a Virtual Machine Runs in Microsoft Azure (Linux & Windows) to Protect your Software when distributed via the Azure Marketplace

mszCool [mario] 4/16/2015 1:56:00 AM

Since the introduction of the Azure Marketplace many publishers are trying to protect their software from being taken off Virtual Machines that have been created through the Azure Marketplace since that would allow customers to run the software without being charged for it. In this blog-post I do show several options to detect in your software, whether a Virtual Machine runs in Azure or not. Using all of them together or any combination is possible as well, of course. Read more on my blog: http://blog.mszcool.com/index.php/2015/04/detecting-if-a-virtual-machine-runs-in-microsoft-azure-linux-windows-to-protect-your-software-when-distributed-via-the-azure-marketplace/
