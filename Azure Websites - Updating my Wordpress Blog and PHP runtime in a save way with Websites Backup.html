As some of my readers know my blog runs on Microsoft Azure Websites. Since I launched this new version back in summer 2014 I literally had no maintenance tasks to do. But about half a year ago I started getting notifications that PHP 5.3 will be deprecated on Azure Websites. Since my blog up until know ran Wordpress 3.x with PHP 5.3 I finally had to do something to ensure a smooth transition. If you want to see the detailed (yet simple) step-by-step journey and how cool Azure Websites plays together with Wordpress, PHP and MySQL, then read on here: http://blog.mszcool.com/index.php/2015/01/azure-websites-updating-my-wordpress-blog-and-php-runtime-in-a-save-way-with-websites-backup/
